# Code Style & Conventions

- C# latest features (records, pattern matching, file-scoped namespaces)
- `sealed record` for DTOs and events
- MediatR/Mediator pattern for events/notifications
- Channel-based pipeline architecture for streaming
- XML doc comments on public APIs
- Task-based async throughout (use ValueTask when appropriate)
- Content-addressable naming (hashes as blob names)
- 3-tier caching pattern (L1 memory, L2 disk, L3 remote)
